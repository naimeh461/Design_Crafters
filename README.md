# Design Crafters

* This website is a Course Related related website.
* Here There are 3 type of user student, instructor and admin.
* Student can add course and Pay for the course .
* Instructor can add the course and See his/her Course Status. 
* Admin can Make A user Instructor or Admin.
* The Security For each user Handle By The website.
